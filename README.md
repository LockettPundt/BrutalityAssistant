## Brutality Assistant

Keep track of the jobs you've applied for with this handy assistant.

[brutalityassistant](https://www.brutalityassistant.com)

### Tools

- React
- Grommet
- MongoDB
- Express
- Nodejs
- JSON Web Tokens
- Validator

### To install 


clone this repo

cd into the directory

```
npm i
```

```
npm start
```
